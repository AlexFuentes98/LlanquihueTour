# LlanquihueTour
Proyecto POO - Agencia de Turismo Llanquihue Tour

//proposito

  Este proyecto es con el objetivo de diseñar e implementar un modelo orientado a objetos, que permita organizar y representar de forma estructurada la información de las personas relacionadas con la agencia de turismo "Llanquihue Tour", mediante el uso de clases, herencias, composicion y encapsulamiento. se busca solucionar problemas de organizacion, duplicacion de datos y gestion de registros, sentando las bases para la futura digitalización de los procesos de la presente empresa.
  
// Características

- Encapsulamiento
- Herencia
- Composición
- Constructores
- Getters y Setters
- Método toString()

- Persona
